# Background assets

Each weather theme has three JPEG backgrounds:

```text
rain-0.jpg ... rain-2.jpg
meteor-0.jpg ... meteor-2.jpg
blossom-0.jpg ... blossom-2.jpg
snow-0.jpg ... snow-2.jpg
beach-0.jpg ... beach-2.jpg
```

Keep replacement images in JPEG format and use the same names so the renderer and
smoke test continue to work. Prefer a 16:9 crop, strong negative space, and enough
contrast for white quota text.
